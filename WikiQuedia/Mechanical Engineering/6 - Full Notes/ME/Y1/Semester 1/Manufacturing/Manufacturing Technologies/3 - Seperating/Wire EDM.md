# Wire EDM
### Process
This process is closely related to [EDM](EDM). It's name stands for 'Wire <u>E</u>lectrical <u>D</u>ischarge <u>M</u>achining'. In this process a wire moves through the metal like a [saw](mechanical%20seperating%20processes#sawing) would, but without the reciprocating motion. Instead sparks form plasma between the wire and the [workpiece](Manufacturing%20Technologies%20Hub#Terms%20and%20Disambiguation). The wire shoots electricity between itself and the workpiece through an electrically conductive fluid. Where the electricity meets the workpiece plasma manifests and a small bit of metal is parted from the workpiece, which is removed through the fluid, and the cycle repeats.
For this to work the workpiece has to be submerged entirely in an electrically conductive fluid.

### Characteristics
This process is for __conductive metals__ only.
##### Positive
- This process leaves <u>no</u> [heat affected zone](Crystal%20Manipulation%20and%20Deformation#hot%20deformation).
- This process leaves very smooth edges.
- This process can be very precise.
- This process can create very complicated shapes.
- This process also works for blocks of metal, not just sheets.
- This process has no tool wear
##### Negative
- This process only works for conductive metals.
- This process goes very slow.
